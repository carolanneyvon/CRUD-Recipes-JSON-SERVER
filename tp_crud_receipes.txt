Instructions
Création d'un système interactif de recettes de cuisine

La mission : Vous devez créer un CRUD ( Create Read Update Delete) qui permette de gérer des recettes de cuisine. Le Back end est géré par json-server. Vous devez créer la partie front de l'application : le HTML CSS JS. Vous êtes libre d'utiliser un framework CSS ou pas. Vous pouvez utiliser par exemple Bootstrap ou TailwindCSS.  

En revanche, vous ne pouvez utiliser uniquement du Vanilla JS (pas de framework JS).

Pour lancer le serveur : 
1- Installer json-server :  npm install -g json-server
2- Mettre en place le fichier db.json dans le dossier 
3- Démarrer le serveur à l'aide de la commande json-server --watch db.json

Accéder au données via l'URL : http://localhost:3000/receipes

Exemple de CRUD via l'API : https://jsonplaceholder.typicode.com/guide/

Le travail est à rendre dans l'onglet devoir de Teams via un lien github ou gitlab (n'oubliez pas de mettre le repo en public pour que je puisse y accéder).

Enjoy !
